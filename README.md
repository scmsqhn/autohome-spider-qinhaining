## 优化路径
### 目前置信度只有64%,可以从以下几个部分进行优化
* (1) 数据准备尚不够完备,目前只引入传统汽车组件部分,仅需引入电子部分和品牌部分
* (2) 通过分布式,进行运算后,根据置信度权重相加,求得最优解
* (3) 增加EM模型,引入未知部分
* (4) 增加时间模型,对上市时间对车辆售价的影响进行评估
